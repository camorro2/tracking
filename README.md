## 🧩 الأدوات الثلاث

### 1️⃣ AndroGhost — باني APK المخترق

**ما يفعله:**  
يأخذ تطبيقاً حقيقياً (آلة حاسبة، كشاف، مشغل موسيقى) ويحقن فيه بايلود تحكم عن بعد.

**التقنية:**  
- 7 طبقات تشفير Polymorphic (تتغير البصمة كل مرة)
- حقن Smali مباشر في كود التطبيق الأصلي
- توقيع رقمي متغير — كل APK له توقيع مختلف
- التشغيل عند فتح التطبيق + عند تشغيل الجهاز + في الخلفية

**أوامر التحكم:**
| الأمر | الوظيفة |
|-------|---------|
| `shell <cmd>` | تنفيذ أمر في شل الجهاز |
| `cam_capture` | تصوير كاميرا أمامية |
| `cam_capture_back` | تصوير كاميرا خلفية |
| `sms` | قراءة جميع الرسائل |
| `location` | الحصول على الموقع الجغرافي |
| `contacts` | سحب جهات الاتصال |
| `call_log` | سجل المكالمات |
| `mic_record` | تسجيل صوت من الميكروفون |
| `download <path>` | تحميل ملف من الجهاز |
| `upload <url> <path>` | رفع ملف إلى الجهاز |
| `persist` | تثبيت البقاء بعد إعادة التشغيل |
| `self_destruct` | حذف التطبيق نهائياً |

---

### 2️⃣ PulseInject — محرك حقن الملفات

**ما يفعله:**  
يخفي بايلود اختراق داخل ملفات تبدو بريئة — صور، PDF، فيديوهات.

**التقنيات:**
| نوع الملف | التقنية | الوصف |
|-----------|---------|-------|
| **JPEG** | Polyglot | البايلود في Comment Segment + بعد EOI — الصورة تعمل طبيعي |
| **PDF** | JavaScript Stager | OpenAction يشغل الكود تلقائياً عند فتح الملف |
| **MP4** | UUID Box | البايلود في moov/udta metadata — الفيديو يعمل طبيعي |
| **APK** | Fusion | دمج APK مخترق مع APK آخر في تطبيق واحد |

---

### 3️⃣ PhantomLink — محرك الروابط الخبيثة

**ما يفعله:**  
يولّد روابط اختراق متطورة — يصيد، يخترق WebView، ويحمّل الملفات تلقائياً.

**الوضعيات:**
| الوضع | الوصف |
|-------|-------|
| **🌐 Phishing Server** | خادم تصيد بقوالب Google، Facebook، Instagram (متطابقة 99%) |
| **📥 Drive-By Download** | صفحة تحديث وهمية تحمّل APK تلقائياً دون موافقة المستخدم |
| **🕸️ WebView Exploit** | يستغل WebView — يسرق الكوكيز، الجلسات، والملفات المحلية |
